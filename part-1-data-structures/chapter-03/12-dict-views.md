# 字典视图：`keys()` / `values()` / `items()`（§3.8）

> **本篇定位**：《流畅的 Python》**§3.8**：Python 3 中 **`dict.keys()`**、**`dict.values()`**、**`dict.items()`** 返回的不是列表，而是**字典视图**——映射的**动态只读投影**。  
> **相关**：三种映射的 API 表里对 `keys`/`values`/`items` 的勾选见 `06-dict-defaultdict-ordereddict-api.md`；**键视图**与 **`in`** 的交互在写 `__contains__` 时也会遇到，见 `09-str-key-dict-and-dunder-missing.md`。  
> **配套脚本**：`dict_views_demo.py`。

---

## 一、视图是什么

- **`dict_keys`**、**`dict_values`**、**`dict_items`**：由对应方法返回，内部**持有原字典引用**，**不复制**键值数据。  
- **读侧**：可 **`len`**、**`for`** 迭代、（在版本允许时）**`reversed`**；**不是序列**，**无** **`view[i]`** 下标（会 **`TypeError: ... is not subscriptable`**）。  
- **同步**：原字典增删改后，**同一视图对象**上再迭代 / 取长度，反映**最新**状态（不是 Python 2 那种「拷一份列表」的快照）。

---

## 二、与 Python 2 的差异（概念）

| 对比 | Python 2 | Python 3 |
| :--- | :--- | :--- |
| `keys()` 等 | 返回 **list**（一次性拷贝） | 返回**视图**（动态、省内存） |
| 修改 dict 后旧结果 | 列表不变 | 视图随 dict 变 |
| 需要快照 | 当时列表即快照 | 显式 **`list(d.keys())`** 等 |

---

## 三、三种视图的共性

- **可迭代**；**`len(view)`** 与 **`len(dict)`** 一致（实现上直接反映映射规模）。  
- **不可通过视图修改**映射（无 `append`、无 `__setitem__`）；改数据只能动**原 `dict`**。  
- **视图类型**为解释器内部类型：通常**不能**用 **`dict_values()`** 这类构造器自建实例（会得到 **`TypeError`**），只能从 **`dict`** 的方法拿到（见 `dict_views_demo.py`）。

---

## 四、三种视图的差异

| 视图 | 典型用途 | 集合式运算 |
| :--- | :--- | :--- |
| **`dict_keys`** | 遍历键、与另一组键求交并差 | 支持 **`&` `|` `-` `^`** 等（键需可哈希；语义类似集合） |
| **`dict_items`** | 遍历 `(k, v)` | 当 **(键, 值) 元组可哈希**时支持同类集合运算；若值不可哈希会失败 |
| **`dict_values`** | 仅遍历值 | **不**提供与 `dict_keys` 同级别的集合运算 |

---

## 五、`reversed()` 与插入顺序

- **Python 3.7+**：`dict` **保留插入顺序**（语言规范）。  
- **`reversed(d)`**、**`reversed(d.keys())`**、**`reversed(d.values())`**、**`reversed(d.items())`**：在**当前 CPython** 中，对**标准 dict** 的上述视图均可按**逆插入序**遍历（以你使用的 Python 版本文档为准；**3.8** 起对视图可逆性有明确加强）。若需兼容极老版本，以官方文档为准。

---

## 六、实战注意

1. **需要列表快照**时：写 **`list(d.keys())`** / **`list(d.values())`**，不要假设视图已是 list。  
2. **集合运算**：**`d.keys() & other.keys()`** 常比「转 `set` 再算」更直接；**`items`** 运算时注意**值是否可哈希**。  
3. **性能直觉**：遍历视图通常**不**额外复制整张表；需要**重复随机访问**或**下标**时再 **`list(...)`**。

## 八、延伸：与原生 `set` 的关系
