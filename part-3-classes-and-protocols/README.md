# Part 3：类、协议与面向对象设计
本部分深入 Python 面向对象（OOP）的核心设计哲学。

## 核心知识点
- **类与实例**：`__init__`、属性、方法、类方法、静态方法。
- **继承与多态**：子类化、方法重写、多态行为。
- **鸭子类型与协议**：
  - 不依赖 ABC，只实现特定方法（如 `__iter__`）。
  - 可迭代对象 vs 迭代器。
- **抽象基类 (ABC)**：`collections.abc` 模块，定义接口契约。
- **属性管理**：`property`、描述符 (Descriptor)、`__slots__` 内存优化。

## 学习目标
理解 Python 独特的 OOP 模型，能写出符合 Python 风格（Pythonic）的类，并熟练运用协议（Protocol）来定义清晰的接口。
