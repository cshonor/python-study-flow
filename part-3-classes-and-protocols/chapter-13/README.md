# 第 13 章「接口、协议和抽象基类」— 本目录说明

本目录对应《流畅的 Python》（第二版）第 13 章：**接口、协议和抽象基类**。

本章开篇核心观点是 GoF 的那句原则：

> **对接口编程，而不是对实现编程。**

在 Python 语境下，“接口”常常不是 `interface` 关键字，而是以 **协议（protocol）/抽象基类（ABC）/类型提示** 等形式出现。
