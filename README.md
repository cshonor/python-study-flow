# 《流畅的 Python（第 2 版）》学习笔记与练习

这个仓库用于整理《流畅的 Python（第 2 版）》的**笔记**与**配套练习代码**（以“可运行的小实验”为主）。

## 结构说明

- `流畅的Python笔记.md`：汇总笔记（第 1–21 章）+ 4 阶段学习路线
- `stage-1-data-structures/`：第 1–4 章（数据结构基础）
- `stage-2-functions-decorators/`：第 5–8 章（函数与装饰器）
- `stage-3-oop-protocols/`：第 9–12 章（面向对象与协议）
- `stage-4-iterators-concurrency-asyncio/`：第 14–18 章（迭代器 / 并发 / asyncio）

每个阶段目录都包含一个入口 `README.md`，建议用于存放：

- 可直接运行的小实验（一个文件验证一个知识点）
- 从练习中抽出来、可复用的小工具函数/小模块
- 阶段回顾总结（要点 + 易错点 + 你自己的示例链接）

## 使用方式（推荐流程）

- 先在 `流畅的Python笔记.md` 里阅读对应章节要点
- 在相应的 `stage-*` 目录中写“最小可复现”的示例脚本并运行验证
- 尽量让单个脚本足够小、足够聚焦：宁可多个小文件，也别堆一个大文件
- 每个阶段结束后写一页回顾，并列出你写过的示例文件清单

## 命名约定（建议）

- 实验脚本：`ex_###_topic.py`（例如：`ex_010_slicing_steps.py`）
- 可复用代码：`utils_*.py`，或在阶段目录下新建 `utils/` 子目录

## 依赖说明

大部分示例应能在较新的 Python 3.x 环境中直接运行、无需额外依赖。
如果后续引入第三方包，请新增 `requirements.txt` 记录依赖。
