# python-study-flow
《流畅的 Python》(Fluent Python) 系统化学习与笔记仓库。

本仓库按照书籍核心知识点进行模块化重构，分为 5 个核心 Part，循序渐进掌握 Python 高级特性与工程化思维，为后续量化交易、数据分析与 AI 开发奠定底层基础。

## 学习架构
| 模块 | 核心内容 | 关键技能 |
| :--- | :--- | :--- |
| 📚 Part 1 | 数据结构与特殊方法 | 序列、映射、集合，`__len__`、`__getitem__` 等魔术方法 |
| 🧩 Part 2 | 函数作为对象与一等公民 | 高阶函数、闭包、装饰器、函数式编程 |
| 🏗️ Part 3 | 类与面向对象协议 | 自定义类型、继承、鸭子类型、抽象基类 |
| ⚡ Part 4 | 控制流与并发 | 上下文管理器、协程、异步编程、并发模型 |
| 🧠 Part 5 | 元编程与高级技巧 | 装饰器类、元类、描述符、属性拦截 |

```text

├── part-3-classes-and-protocols/  # 类与协议
├── part-4-control-flow/    # 控制流与并发
├── part-5-metaprogramming/ # 元编程与高级技巧
└── 流畅的Python笔记.md      # 综合笔记
```
