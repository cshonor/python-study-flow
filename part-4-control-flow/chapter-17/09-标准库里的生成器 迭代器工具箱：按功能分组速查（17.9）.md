# 标准库里的生成器/迭代器工具箱：按功能分组速查（17.9）

本节目标不是“背完 20+ 个函数”，而是建立一个心智模型：

- **你想对 iterable 做什么操作？**  
  筛选 / 映射 / 合并 / 扩展 / 重排分组 / 组合排列  
  → 去对应组里找工具（很多都在 `itertools`）。

---

## 1) 筛选（filtering）

| 工具 | 模块 | 一句话 |
|------|------|--------|
| `filter(pred, it)` | builtins | 保留 pred 为真的项 |
| `itertools.filterfalse(pred, it)` | itertools | 保留 pred 为假的项 |
| `itertools.takewhile(pred, it)` | itertools | 条件为真就继续，第一次为假立刻停 |
| `itertools.dropwhile(pred, it)` | itertools | 跳过 pred 为真的前缀，之后全部保留 |
| `itertools.compress(data, selectors)` | itertools | selectors 为真才保留对应 data |
| `itertools.islice(it, start, stop, step)` | itertools | 对任意 iterable 做惰性切片 |

---

## 2) 映射/变换（mapping）

| 工具 | 模块 | 一句话 |
|------|------|--------|
| `map(fn, it)` | builtins | 把 fn 应用到每一项 |
| `itertools.starmap(fn, it)` | itertools | 解包参数后再调用 fn |
| `itertools.accumulate(it, fn=...)` | itertools | 累计/前缀归约（和/积/自定义） |
| `enumerate(it, start=0)` | builtins | 生成 `(index, item)` |

---

## 3) 合并/并行（merging）

| 工具 | 模块 | 一句话 |
|------|------|--------|
| `itertools.chain(a, b, ...)` | itertools | 串联多个 iterable |
| `itertools.chain.from_iterable(it)` | itertools | 扁平化“可迭代的可迭代” |
| `zip(a, b, ...)` | builtins | 并行打包（最短者耗尽就停） |
| `itertools.zip_longest(a, b, fillvalue=...)` | itertools | 并行打包（最长者耗尽才停） |
| `itertools.product(a, b, ...)` | itertools | 笛卡尔积（惰性） |

---

## 4) 扩展/无限（expanding / infinite）

| 工具 | 模块 | 一句话 |
|------|------|--------|
| `itertools.count(start=0, step=1)` | itertools | 无穷等差数列 |
| `itertools.cycle(it)` | itertools | 无穷循环 |
| `itertools.repeat(x, times=None)` | itertools | 重复产生同一个值 |
| `itertools.pairwise(it)` | itertools | 相邻重叠对（`(x0,x1),(x1,x2)...`） |

**坑**：无限迭代器不要直接 `list(...)`，用 `islice/takewhile` 截断。

---

## 5) 重排/分组（rearranging / grouping）

| 工具 | 模块 | 一句话 |
|------|------|--------|
| `itertools.groupby(it, key=...)` | itertools | 相邻分组（通常先排序） |
| `reversed(seq)` | builtins | 反向迭代（需要序列或实现 `__reversed__`） |
| `itertools.tee(it, n=2)` | itertools | 复制迭代器（注意缓存开销） |

`groupby` 的关键点：**按相邻元素分组**，想按 key 全局分组通常先 `sorted(..., key=...)`。

---

## 6) 组合/排列（combinatorics）

| 工具 | 模块 | 一句话 |
|------|------|--------|
| `itertools.permutations(it, r)` | itertools | r 长度排列（有顺序） |
| `itertools.combinations(it, r)` | itertools | r 长度组合（无顺序） |
| `itertools.combinations_with_replacement(it, r)` | itertools | 允许重复的组合 |
