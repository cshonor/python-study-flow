# Part 4：控制流、并发与异步编程
本部分专注于程序执行流程与高性能并发处理。

## 核心知识点
- **上下文管理器**：`with` 语句，`__enter__` / `__exit__` 资源管理。
- **并发模型**：
  - 多线程 (Threading)：适用于 I/O 密集型。
  - 多进程 (Multiprocessing)：适用于 CPU 密集型。
- **异步编程 (Asyncio)**：
  - `async/await` 语法。
  - 事件循环与协程 (Coroutine)。
  - 异步任务队列。
- **可迭代对象与生成器**：`yield`，生成器表达式，惰性序列。

## 学习目标
掌握 Python 高级控制流结构，能处理大规模数据（如量化回测中的海量K线数据），并编写高性能的异步 I/O 服务。
