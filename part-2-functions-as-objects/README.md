## 核心知识点
- **一等对象**：函数是对象，可作为参数、返回值、存储在数据结构中。
- **高阶函数**：`map()`、`filter()`、`sorted()`、`functools.reduce()`。
- **闭包 (Closure)**：自由变量的延迟绑定，保持函数状态。
- **装饰器 (Decorator)**：
  - 函数装饰器与类装饰器。
  - 叠放装饰器 (@wrap)。
  - 带参数的装饰器。
- **函数式工具**：`functools` 模块，`partial` 偏函数，`singledispatch` 泛函数。
