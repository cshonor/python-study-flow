# 第 7 章「函数是一等对象」— 本目录说明

本目录对应《流畅的 Python》（第二版）**第 7 章：函数是一等对象**。

本章回答的核心问题是：**在 Python 里，函数和普通数据一样，都是运行时存在的对象**——可以绑定到名字、放进容器、当作参数传入、当作返回值传出。这是高阶函数、闭包、装饰器等机制的公共地基。
