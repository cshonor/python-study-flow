# 示例 7-1 / 7-2：函数是对象、可赋值、可传给 `map`（高阶函数）

这一篇对应书里用交互式例子把「函数是一等对象」**落到可运行的代码上**：先看函数作为**普通对象**有哪些属性，再看**赋值别名**和 **`map(函数, 可迭代对象)`**。

配套脚本：`01_first_class_functions_demo.py`（开头三段对应书中 7-1、7-2 与常见属性演示）。

---

## 一、示例 7-1：函数是运行时存在的对象，且有属性

```python
def factorial(n: int) -> int:
    """returns n!"""
    return 1 if n < 2 else n * factorial(n - 1)
```

你可以像检查任何对象一样检查它：

- **`factorial.__doc__`**：文档字符串（没有写三引号文档时多为 `None`）。
- **`type(factorial)`**：用户定义函数通常是 **`function`**（内置函数则是 `builtin_function_or_method` 等）。
- **`factorial.__class__`**：与 `type(factorial)` 在通常用法下一致，都指向「这个对象所属的类」。

**结论**：用户写的 `def` 函数，在 CPython 里就是 **`function` 类型的一个实例**——所以才有 `. __doc__`、`. __name__` 等属性，和「函数是数据」不矛盾。

## 二、示例 7-2：别名赋值 + 把函数交给 `map`

### 2.1 赋值：多一个名字指向同一个函数对象

```python
fact = factorial
assert fact(5) == factorial(5)  # 成立 → 别名与原名同一函数，5! 均为 120
```

`fact` 与 `factorial` 是同一函数对象的**两个引用**（一般 `fact is factorial` 为 `True`）。

### 2.2 `map(factorial, range(11))`：高阶函数

`map` 的语义是：**对可迭代对象里每一个元素，调用你传入的那个函数**，返回一个迭代器（在 Python 3 里需要 `list(...)` 才能一次性看到所有结果）。

```python
list(map(factorial, range(11)))
```

这里：

- 第一个参数必须是 **callable**（可调用对象）；
- `factorial` 作为「数据处理规则」被传进去——这就是**高阶函数**：接收别的函数当参数。

**和列表推导式的关系**：`[factorial(n) for n in range(11)]` 语义接近，Often 在 Python 里推导式更直观；`map` 在「已经有一个现成的一元函数」时很简洁。

## 三、函数对象上常见属性（查阅与元编程时会用到）

| 属性 | 含义 |
|------|------|
| `__name__` | 定义时的名字（字符串） |
| `__doc__` | 文档字符串 |
| `__defaults__` | 默认参数值的元组（仅位置/关键字默认值，与参数从左到右对应） |
| `__kwdefaults__` | 仅限关键字参数的默认值字典，可能为 `None` |
| `__code__` | 代码对象（字节码、变量名、常量表等；深入 CPython 才常看） |
| `__annotations__` | PEP 484 风格注解（参数名 → 类型对象；可能含 `'return'`） |

这些属性**不影响**「函数能不能调用」；它们描述的是**函数对象自身的元数据**。

---

## 四、「可调用对象」（Callable）不止 `def` 一种

凡是可以写成 `obj(...)` 的，都是可调用对象，常见来源包括：

- **`def` / `lambda`**：用户函数对象；
- **内置函数**：如 `len`、`print`（实现细节因解释器而异）；
- **类型对象（类本身）**：`MyClass()` 会调用类的调用逻辑并构造实例；
- **实现了 `__call__` 的实例**：实例像函数一样被「调用」；
- **方法**：绑定方法、未绑定方法等（本质仍是 callable）。

判断「能不能当函数用」可用内置函数 **`callable(obj)`**。

## 五、与后续小节的关系（本章后文会展开）

- **装饰器**：依赖「函数可作为参数/返回值」；
- **闭包**：内层函数仍是函数对象，但携带了外层作用域的绑定；
- **`functools`**：如 `partial`（基于已有函数造新函数）、`lru_cache`（在函数对象上挂缓存）。

本篇只建立「函数是对象、能传给 `map`」的实感，后面再逐节加深。
