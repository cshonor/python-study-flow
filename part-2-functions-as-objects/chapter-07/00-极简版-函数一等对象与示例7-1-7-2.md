一页式速览；展开见 `01-第 7 章开篇：函数作为「一等对象」是什么意思.md`、`02-示例 7-1 7-2：函数是对象、可赋值、可传给 map（高阶函数）.md`。

---

## 核心一句话

**Python 里函数就是对象**：可赋值、当参数、当返回值、进容器 → **一等对象**。

---

## 示例 7-1（照抄运行）

```python
def factorial(n: int) -> int:
    """returns n!"""
    return 1 if n < 2 else n * factorial(n - 1)

print(factorial.__doc__)
print(type(factorial))
print(factorial.__name__)
```

---

## 示例 7-2（别名 + map）

```python
fact = factorial
print(fact(5))  # 120

print(list(map(factorial, range(6))))
# [1, 1, 2, 6, 24, 120]
```

Python 3：`map` 返回迭代器，常用 `list(...)` 看全结果。等价：`[factorial(n) for n in range(6)]`。
